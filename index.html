Proyecto actual de trabajo (Pixelvania)

Descripción:

Pixelvania es un videojuego de plataformas 2D que estoy desarrollando. Combina gráficos retro en pixel art con mecánicas clásicas, ofreciendo niveles desafiantes llenos de enemigos, secretos y aventuras.

Objetivo del proyecto:

Crear un videojuego divertido y desafiante que combine mecánicas clásicas de plataformas con elementos de exploración.

Herramientas y tecnologías utilizadas


Lenguaje de programación: C#

Motor de desarrollo: Unity

Diseño gráfico: Libresprite para pixel art

Sistema Operativo: Windows

Editor de código: Visual Studio/VS Code
